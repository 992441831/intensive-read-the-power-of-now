# 精读《当下的力量》 · 系列专题报告

> 回到托利的概念内核，解剖这个时代的精神内耗。
>
> "你不等于你的大脑。"

本仓库是一个「精读《当下的力量》」专题报告系列：回到埃克哈特·托利《当下的力量》（曹植译，中信出版社）的概念地基——思维认同、小我、痛苦之身、心理时间、临在、臣服、内在身体——用它们分析精神内耗、注意力碎片化、焦虑、失眠等当代困境。精读而不迷信：每篇附批判性反思与可立刻实践的练习。
